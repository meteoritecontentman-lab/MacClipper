ℹ️ Setup Instructions

  1. Click "Generate API Token" above
  2. Copy the token that appears (it will only be shown once)
  3. Open MacClipper on your Mac
  4. Go to Settings → Share + Storage → API Token
  5. Paste your token and save
  6. Click the cloud icon on any clip to upload it
